| Version | Supported          |
| -------- | ------- |
| Windows Server 2025 | ❔ |
| Windows Server 2022 | :white_check_mark:  |
| Windows Server 2019 | :white_check_mark: |
| Windows Server 2016 | :white_check_mark: |
| Windows Server 2012 R2  | :white_check_mark: |
